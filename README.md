# LootRememberer

_World of Warcraft addon made for official client (Burning Crusade Anniversary edition)._

Right click on need, greed, or pass to remember the selection for that item and auto-apply it in future on that character.
(Hold shift to save it to account instead of character.)

Type /lootrememberer or /lr to browse and delete saved records.

This addon now uses only the default WoW API and has no external library dependencies.
